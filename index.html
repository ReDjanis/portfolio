Веб-сайт: «Ножевая мастерская Александра Трусова» (в разработке).

Примененные технологии: scss, native js, jquery (для slick slider), gulp.

Задача: сделать информативный сайт, на котором пользователи могли бы ознакомиться с услугами мастера, условиями заказа товаров, а также просмотреть каталог этих товаров. На сайте нет возможности заказа, но предусмотрена форма обратной связи.

Особенности: в html на весь сайт создана одна основная страница с несколькими тегами main. Каждому тегу main соответствует своя страница на сайте. Активная страница – это включение соответствующего тега main и скрытие остальных (путем добавления атрибута hidden). Это сделано для ускорения работы сайта при переключении между страницами. При обновлении сайта пользователь останется на текущей «странице», так как информация об активном теге main сохраняется в sessionStorage.
Весь каталог товаров хранится непосредственно в корне проекта (не на сервере). Загрузка всех карточек товаров на страницу сделана через javascript. В каталоге реализована возможность сортировки товаров, пагинация, кнопка «Показать еще». Также реализована возможность поиска по каталогу.
Для ускорения загрузки изображений в тег img добавлены атрибуты width и height.
Адаптив, форма обратной связи, страница 404 - пока в разработке.
